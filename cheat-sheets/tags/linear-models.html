Linear Regression

Fits a linear model with coefficients w=(w1,…,wp)w = (w1, …, wp)w=(w1,…,wp) to minimize the residual sum of squares between the observed targets in the dataset, and the targets predicted by the linear approximation.

  • model equation: y=b0+b1x1+b2x2+...+bpxpy = b_0 + b_1x_1 + b_2x_2 + ... + b_px_py=b0​+b1​x1​+b2​x2​+...+bp​xp​
from sklearn.linear_model import LinearRegression
lr = LinearRegression()

Ridge Regression

Applies L2 regularization to reduce the complexity of the model and prevent overfitting.

  • model equation: y=b0+b1x1+b2x2+...+bpxp−α∑j=1pbj2y = b_0 + b_1x_1 + b_2x_2 + ... + b_px_p - \alpha \sum_{j=1}^{p} b_j^2y=b0​+b1​x1​+b2​x2​+...+bp​xp​−α∑j=1p​bj2​
  • if α=0\alpha = 0α=0, then the model is the same as Linear Regression
  • if α=∞\alpha = \inftyα=∞, then all coefficients are zero, and the result is a constant mean
from sklearn.linear_model import Ridge
ridge = Ridge(alpha=1.0)
